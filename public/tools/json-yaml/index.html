JSON ↔ YAML 转换

一站式将 JSON 转换为 YAML，或将 YAML 转换为 JSON，自动保留缩进格式并校验语法，适用于配置文件与后端响应调试。

打开交互式转换器

功能亮点

  • 双向转换：JSON → YAML 与 YAML → JSON 均可一键完成。
  • 错误定位：语法不合法时会即时提示具体行列。
  • 格式安全：支持多行字符串、注释、特殊字符。
  • 结果导出：可以复制到剪贴板或下载为 .json / .yaml 文件。

使用步骤

  1. 输入待转换的 JSON 或 YAML 内容，另一侧可留空。
  2. 点击相应的「转换」按钮，立即得到格式化结果。
  3. 如提示语法错误，按照提示位置修正后再次转换。
  4. 可直接复制结果，或下载为需要的文件格式。

常见问题

工具是否联网使用？

不需要联网，所有数据仅在浏览器内存中处理。

支持自定义缩进吗？

JSON 输出默认为 2 空格缩进，后续版本将支持自定义选项。

如何在英文或日文界面下使用？

进入交互式转换器后，可在页面右上角切换语言或通过附带的 ?lang 参数访问。
