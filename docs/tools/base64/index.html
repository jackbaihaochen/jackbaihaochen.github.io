Base64 编解码工具

高效转换文本与图片的 Base64 编码/解码工具，支持批量复制、即时预览与文件下载，适合开发调试与内容发布。

进入交互式工具

功能亮点

  • 文本互转：将字符串编码为 Base64 或解码回原文。
  • 图片支持：上传 PNG/JPEG/SVG 等图片，可生成 Base64 字符串并预览还原效果。
  • 零上传：所有操作在浏览器本地完成，保障敏感数据安全。
  • 便捷导出：一键复制结果或下载图片文件，支持多语言界面。

操作步骤

  1. 在左侧文本框输入或粘贴原文，点击「编码」即可得到 Base64 字符串。
  2. 反向解码时，在右侧粘贴 Base64 内容，点击「解码」即可恢复原文。
  3. 若需处理图片，可点击「选择图片」或粘贴 Base64 字符串，支持即时预览校验。
  4. 处理完成的结果可复制到剪贴板，或下载为文件以便分享。

常见问题

Base64 编码是否安全？

Base64 仅用于文本包装或传输，并非加密；若需保密，请另行加密数据。

可以处理多大的文件？

受限于浏览器内存与取值限制，一般建议不超过 5MB 的图片或文本。

如何切换语言？

在交互式工具右上角可切换中文、英文与日文，也可通过 URL 的 ?lang= 参数直接访问。
