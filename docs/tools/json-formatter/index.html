JSON 格式化（在线）

浏览器本地运行的免费 JSON 格式化工具。一键美化或压缩大文件，自动检测语法错误，并支持复制、下载、粘贴回源代码。

立即打开交互式工具

功能亮点

  • 本地处理：不会上传到服务器，满足隐私与安全要求。
  • 多种操作：支持美化缩进、压缩、语法校验与错误行列提示。
  • 便捷分享：结果可一键复制或下载为 JSON 文件。
  • 多语言界面：支持中文、英文、日文切换。

使用指引

  1. 在工具页面左侧粘贴或输入待处理的 JSON 文本。
  2. 点击「美化」生成易读格式，或点击「压缩」用于接口传输。
  3. 若存在语法问题，界面会提示错误位置；修正后再次操作即可。
  4. 处理完成后，可直接复制结果或下载为 formatted.json。

常见问题

我的数据会被上传或记录吗？

不会。所有操作均在浏览器内存中完成，刷新页面即可清除。

支持多大的 JSON 文件？

常见的配置文件或 API 响应都能即时处理；极大的文件取决于本地浏览器内存。

为什么结果里包含中文字符？

默认不会转义中文，便于人工阅读；可在压缩结果中获得仅 ASCII 的输出。
